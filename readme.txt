Meeting booking form with fixed 30-minute slots for event scheduling.

== Description ==
Provides shortcode [davion_meetings_form] for event scheduling with:
- Fixed slot windows
- Server-side slot locking
- Confirmation emails
- ICS attachments for calendar import

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/.
2. Activate the plugin in WordPress.
3. Add shortcode [davion_meetings_form] to any page.
